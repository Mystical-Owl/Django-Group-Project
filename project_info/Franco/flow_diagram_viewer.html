Flow Description

Primary Sequence

  1. Customer → User Account (registration)
  2. User Account → Decision (choose investment path)
  3. Decision branches into two paths:
    • Direct Path: Decision → Investment (red arrow)
    • Questionnaire Path: Decision → Questionnaire → QuestionAnswer → Risk Assessment → Affordability Score → Product → Investment (green arrow)
  4. Investment → Portfolio (investment added to portfolio)
  5. Portfolio → Daily Report (performance reporting)
  6. Product → Portfolio (products are included in portfolios)

Entity Definitions
